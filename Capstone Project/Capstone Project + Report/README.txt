LIBRARIES USED :

1.) PANDAS : Pandas is a data analysis library of python. It serves for the purpose of wranging, analysis, visualization for the tabular datasets.  


2.) NUMPY : Numpy is the python library for numerical computation. Pandas is built on the top of numpy. Numpy makes working with numerical vectors and arrays  of data a lot easier.


3.) MATPLOTLIB : Matplotlib is the fundamental visualizing library of python. It is used for generating fundamental plots such as barplot, histograms, scatterplots etc. 


4.) SEABORN : Seaborn is primarily used for statistical plotting and is built over python's matplotlib. This can generate plots in a very few lines than as compared to generating the same plot using matplotlib.
